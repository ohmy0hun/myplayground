# 암호화폐 알고리즘 트레이딩 봇

Upbit을 통한 자동 매매 시스템 구축 프로젝트

## 📋 프로젝트 개요

- **목표**: 알고리즘 트레이딩을 통한 암호화폐 자동매매 시스템 구축
- **거래소**: Upbit (업비트)
- **전략**: 이동평균 전략 + RSI 전략 (비교 분석)
- **개발 언어**: Python

## 🎯 진행 단계

1. **Phase 1**: 데이터 수집 및 API 연동 (1-2주)
2. **Phase 2**: 전략 개발 및 백테스팅 (2-3주)
3. **Phase 3**: 페이퍼 트레이딩 (1-2주)
4. **Phase 4**: 실전 트레이딩 (소액, 신중하게)
